Glasscribe is a speech-to-text app for Mac. It runs on the device, stays offline, and shows live subtitles as you speak.

## 🔧 Troubleshooting

If Glasscribe does not work as expected, check these items:

- Make sure microphone access is allowed in macOS settings
- Check that the right input device is selected
- Confirm the app is open and active
- Restart the app if subtitles stop updating
- Make sure your language choice matches what you are speaking
- Close other apps that may use the microphone

If the app still does not respond, quit it and open it again.
